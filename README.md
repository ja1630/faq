# faq

A simple FAQ (Frequently Asked Questions) web page built using HTML, CSS, and JavaScript.

---

## 📌 Features

- Clean and simple FAQ layout
- Click to expand/collapse answers
- Beginner-friendly JavaScript interactivity
- Responsive design (works on mobile and desktop)

## 🛠️ Built With

- HTML
- CSS
- JavaScript
